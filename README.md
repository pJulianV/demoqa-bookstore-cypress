# DemoQA Bookstore Cypress

Este proyecto contiene pruebas automatizadas para la DemoQA Book Store usando Cypress.

## Requisitos
- Node.js instalado
- npm instalado

## Instalación
1. Abre una terminal en la carpeta del proyecto.
2. Ejecuta:
   
   npm install

Esto instalará todas las dependencias necesarias.

## Ejecución de pruebas
Para abrir la interfaz gráfica de Cypress y ejecutar las pruebas:

    npx cypress open

Para ejecutar todas las pruebas en modo consola:

    npx cypress run
